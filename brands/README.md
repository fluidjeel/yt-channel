# Brand DNA (Layer 3)

Distinct **voice, look, and pacing** on top of the same niche + core.

Prevents every channel from sharing one creator fingerprint.

## Formula

```
Content = Core + Niche Profile + Brand DNA
```

Examples:

| Core | Niche | Brand | Result |
|------|-------|-------|--------|
| ✓ | emotional_healing | whisprs_style | Poetic lo-fi healing Shorts |
| ✓ | tech_reviews | neural_brief | AI news review channel |
| ✓ | horror | dark_lantern | Faceless horror Shorts |
